🔧 راهنمای API

بازی «پازل جعبه‌چین» از endpoint زیر برای دریافت آگهی تصادفی استفاده می‌کند. وقتی آگهی فعالی موجود نباشد، بازی به‌طور خودکار از یک تبلیغ پیش‌فرض استفاده می‌کند.

دریافت آگهی تصادفی

GET https://chabooksaz.ir/api/ads/random

پاسخ موفق:

{ "title": "ساخت سایت اختصاصی", "description": "سایت خود را سفارشی بسازید...", "image_url": "https://chabooksaz.ir/ads/site.jpg", "link_url": "https://chabooksaz.ir/order", "cta_text": "سفارش دهید" }

ثبت کلیک (اختیاری)

POST https://chabooksaz.ir/api/ads/click Content-Type: application/json { "link_url": "https://chabooksaz.ir/order" }

مدیریت آگهی‌ها (نیاز به توکن)

# لیست همه آگهی‌ها GET /api/admin/ads Authorization: Bearer YOUR_TOKEN # ساخت آگهی جدید POST /api/admin/ads Authorization: Bearer YOUR_TOKEN Content-Type: application/json { "title": "...", "description": "...", "link_url": "...", "cta_text": "...", "image_url": "...", "category": "website", "active": true } # ویرایش PUT /api/admin/ads/{id} # حذف DELETE /api/admin/ads/{id}
