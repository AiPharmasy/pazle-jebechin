## 📱 تست APK قبل از انتشار

قبل از ارسال به مارکت، APK را روی گوشی خود تست کنید:

### روش ۱: انتقال مستقیم
1. فایل `pazle-jebechin-v1.2-release.apk` را به گوشی منتقل کنید (بلوتوث/ایمیل/تلگرام/USB)
2. در گوشی: Settings → Security → Unknown Sources را فعال کنید
3. فایل APK را در فایل‌منیجر پیدا کرده و نصب کنید

### روش ۲: ADB
```bash
adb install pazle-jebechin-v1.2-release.apk
```

## 🔄 انتشار نسخه جدید (بعداً)

وقتی تغییراتی در بازی دادید و خواستید نسخه جدید منتشر کنید:

1. کد را تغییر دهید
2. در فایل `android/app/build.gradle`، `versionCode` را افزایش دهید (مثلاً ۳ → ۴)
3. `versionName` را به‌روزرسانی کنید (مثلاً "1.2" → "1.3")
4. APK جدید بسازید:
   ```bash
   cd /home/z/my-project
   bun run build
   npx cap sync android
   cd android
   JAVA_HOME=/home/z/jdk ANDROID_HOME=/home/z/android-sdk ./gradlew assembleRelease
   ```
5. APK جدید را در پنل مارکت آپلود کنید (همان packageName، versionCode بالاتر)
